# MCQ Generator – Hack Node India Project

An interactive web-based MCQ Generator built using **Next.js** and **Tailwind CSS**.  
This tool allows users to input text content and automatically generate category- and difficulty-based multiple-choice questions with instant score evaluation.

Developed as part of **Hack Node India 🏆**.

## Features

-  Select category and difficulty level
-  Paste custom text content
-  Automatically generate multiple types of MCQs
-  Real-time quiz attempt system
-  Instant scorecard after quiz completion
-  Clean and responsive UI design

## Tech Stack

### Frontend
- Next.js (React Framework)
- TypeScript
- Tailwind CSS

### Core Concepts Used
- Dynamic rendering
- State management
- Conditional UI rendering
- Text processing & structured content handling
